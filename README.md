# Text Parser (Processador de Texto)

Este projeto foi criado como um estudo de ferramentas de processamento de texto e de criação de interfaces. Funciona basicamente como um terminal, em que o usuário envia comandos em texto, que são analisados pelo script e retornam uma mensagem ou executam uma função dentro do programa.
